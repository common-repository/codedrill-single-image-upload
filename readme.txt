== Description ==

This plugin will allow to upload an image as attachment. And you will get attachment id of the image. <b> Shortcode: [CD_Single_IMAGE_UPLOAD]</b>. It will create thumbnails as defined in wordpress configuration.

== Installation ==

Upload the CodeDrill Single Image Upload plugin to your blog, Activate it. you can use [CD_Single_IMAGE_UPLOAD] to check the demo. It will automatically create a page with name "Upload Page Demo"

1, 2, 3: You're done!
